# RazExpress

## 해외 상품 구매 주문서 생성 서비스

현지 통화로 표시된 해외 상품 가격에 실시간 환율을 적용해 원화로 환산하고, 총 결제 금액을 주문서 형태로 보여주는 서비스입니다.

### 기능 요구 사항
- [x] `주문 번호`, `외국 통화 종류`, `외국 통화 기준 결제 금액`을 전달받아 `Payment`를 생성한다.
  - [x] `적용 환율`은 https://open.er-api.com/v6/latest 실시간 환율 정보를 이용하여 `원화 환산 금액`을 책정한다.
  - [ ] `적용 환율` API 요청을 일정 시간동안 캐싱하도록 한다.
  - [ ] `원화 환산 금액 유효 시간` 동안만 현재 적용된 환율을 유지한다.
  
- [x] `Payment` 객체를 반환하는 `PaymentService.prepare()` 메서드를 설계해야 한다.
